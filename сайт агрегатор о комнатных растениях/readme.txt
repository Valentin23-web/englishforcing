1. Получил место на бесплатном хостинге.
2. Настроил Apache, чтобы все запросы шли на index.php
3. Создел класс, который при GET-запросе на "/run" подключается к сервису Яндекс XML и, подставляя
названия популярных комнатных растений в запрос, получал первые 20 статей, посвященных этому цветку
в выдаче Яндекса. Статьи записываются в таблицу бд Mysql, с указанием того, к какому растению они относятся,
ссылкой на источник и рядом других полезных параметров. Так как в Яндекс XML есть ограничение по запросам в час, то к http://f90577wq.beget.tech/run
раз в час обращается планироващик задач open server. Чтобы сохраненными в базу оказались статьи по всем растениям из массива. 
4. Класс вывода из базы, хранящийся в папке Model, при обращении на соответствующее url, например http://f90577wq.beget.tech/zamioculcas
выдает из фронтовой части сайта страницы, на которых содержится текст всех спаршенных парсером статей по определенному цветку с указанием на источник.
5. Во View организован вывод кратких содержаний статей на главную страницу с пагинацией, так как, что отдельные статьи можно открыть через homepage.
